# FurryGX Browser

A desktop browser for gamers and furries with an Opera GX / Airâ€“style neon dark UI, Google search, and a slide-out gaming & furry news panel.

## Requirements

- Node.js (LTS is recommended)
- Internet connection (for web browsing and Google search)

## Install

From a terminal in this folder:

```bash
npm install
```

This installs Electron using the version declared in `package.json`.

## Run

From the same folder:

```bash
npm start
```

This will open the FurryGX Browser window.

## Features

- Google-based address/search bar
- Neon dark UI loosely inspired by Opera GX/Air
- Left edge button that slides open a news panel
- Static curated gaming and furry-flavored tech news text
- Bottom status bar with a short tagline about gamers and furries
